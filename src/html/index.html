Getting projects
When loaded the application will show all of the groups your GitLab user can access under the groups dropdown at the top menu. At this time all the projects that you are a member of are also shown under the projects dropdown.
Repository Navigation
  • Once a project has been selected wait a moment for the repository to clone then the repository file structure will appear within the repository browser section.
  • When a folder is selected all of the folders and file beneath it will be copied to your destination in this case the folder structure will remain otherwise the files without any directory structure will be copied into the destination.
Comments
